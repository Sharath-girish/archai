Directory Structure¶

├── archai
│   ├── cifar10_models
│   ├── common
│   ├── darts
│   ├── data_aug
│   ├── nas
│   ├── networks
│   ├── petridish
│   ├── random
│   └── xnas
├── archived
├── confs
├── dockers
├── docs
├── scripts
├── setup.py
├── tests
└── tools
    ├── azure

Most of the functionality resides in the archai folder. nas contains algorithm-agnostic infrastructure that is commonly used in NAS algorithms. common contains common infrastructure code that has no nas-specific code but is infrastructre code that gets widely used. Algorithm-specific code resides in appropriately named folder like darts, petridish, random, xnas

scripts contains entry-point scripts to running all algorithms.
